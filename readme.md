# üé§ Voice Transcriber Pro

A powerful Streamlit application that transcribes audio files and generates AI-powered summaries using Groq's Whisper API and Google's Gemini API.

## ‚ú® Features

- **Fast Audio Transcription**: Powered by Groq's Whisper-large-v3 model
- **AI Summarization**: Generate concise summaries using Google Gemini
- **Multiple Audio Formats**: Supports MP3, WAV, M4A, FLAC, MP4, MPEG, MPGA, WebM
- **Real-time Statistics**: Track word count, compression ratio, and reading time saved
- **User-friendly Interface**: Clean, intuitive UI with audio playback
- **Export Results**: Download transcriptions and summaries as text files

## üöÄ Quick Start

### Prerequisites

- Python 3.8+
- Groq API key (free at [console.groq.com](https://console.groq.com/))
- Google Gemini API key (free at [makersuite.google.com](https://makersuite.google.com/))

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd voice-transcriber-pro

# Install dependencies
pip install streamlit google-generativeai groq langchain-google-genai
```

### Run the App

```bash
streamlit run app.py
```

## üìñ How to Use

1. **Configure API Keys**: Enter your Groq and Gemini API keys in the sidebar
2. **Upload Audio**: Choose an audio file (up to 25MB)
3. **Transcribe**: Click "Transcribe with Groq" to convert speech to text
4. **Summarize**: Click "Summarize with Gemini" to generate a concise summary
5. **Download**: Export your transcription and summary as text files

## üîë Getting API Keys

### Groq API (Transcription)
- Visit [console.groq.com](https://console.groq.com/)
- Sign up for free
- Create an API key
- Free tier: 14,400 requests/day

### Google Gemini API (Summarization)
- Visit [makersuite.google.com](https://makersuite.google.com/)
- Sign in with Google
- Generate an API key
- Free tier: 15 req/min, 1,500 req/day

## üõ†Ô∏è Technologies

- **Streamlit**: Web interface
- **Groq Whisper API**: Audio transcription
- **Google Gemini API**: Text summarization
- **LangChain**: AI framework integration

## üìä Features in Detail

- **Adjustable Summary Length**: Control summary word count (50-500 words)
- **Processing Time**: View transcription and summary generation times
- **Compression Metrics**: See how much the summary condenses the original text
- **Reading Time Calculation**: Estimate time saved by reading the summary

## üîí Privacy

- Audio files are sent to Groq for transcription
- Transcribed text is sent to Google Gemini for summarization
- No data is stored locally by the application
- Review Groq and Google's privacy policies for details
